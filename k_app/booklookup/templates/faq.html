Mendeley cannot export citations into an excel file yet, but there is simple a way around this.
  • Download and install the open source reference manager JabRef http://www.jabref.org/
  • Export your citations from the Mendeley Desktop as a BibTex (*.bib) file
    • File
    • Export
  • Open JabRef and import the .bib file
    • File
    • Import
  • You may have to edit the citations a bit
  • After editing, save as a .csv file from JabRef
    • File
    • Export
  • Open your new .csv file in excel
